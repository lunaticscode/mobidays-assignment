### 3.1 글로벌 필터(GlobalFitler.tsx)

![global-filter](./images/global-filter.png)

- 필터 조건 변경 시, 하단의 **모든 차트와 테이블 데이터가 실시간으로 동기화**되어야 합니다.
- 각 필터 조건은 **AND** 조합으로 적용됩니다.

| **필터**                                | **세부사항**                      | 제약사항                      |
| --------------------------------------- | --------------------------------- | ----------------------------- |
| **집행 기간**                           | 선택한 기간 내 집행 중인 캠페인과 |
| 해당 기간의 일별 데이터를 기준으로 집계 | 초기값(당월 1일~말일)             |
| **상태**                                | 진행중/일시중지/종료              | 초기값(전체) / 다중 선택 가능 |
| **매체**                                | Google / Meta / Naver             | 초기값(전체) / 다중 선택 가능 |
| **초기화**                              | 모든 조건을 초기값으로 복구       |                               |
